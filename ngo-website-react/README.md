## Overview

Welcome to the **NGO Website React** project! This repository hosts a modern, responsive, and completely free React.js website template tailored specifically for Non-Governmental Organizations (NGOs), charities, and non-profits.

Our goal is to help NGOs establish a strong online presence quickly and effortlessly, allowing them to focus more on their core missions and less on technical overhead.

## Tech Stack & UI

This template is built with modern web technologies to ensure a fast, accessible, and easily customizable experience:

- **Framework:** [React 19](https://react.dev/) with [Vite](https://vitejs.dev/)
- **Routing:** [TanStack Router](https://tanstack.com/router) & [TanStack Start](https://tanstack.com/start)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **UI Components:** [Radix UI](https://www.radix-ui.com/) primitives (shadcn/ui architecture)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Forms & Validation:** [React Hook Form](https://react-hook-form.com/) & [Zod](https://zod.dev/)
- **Language:** [TypeScript](https://www.typescriptlang.org/) for robust type-safety

## Features

- 🎨 **Modern Design:** Clean and accessible UI components built with Radix UI and Tailwind CSS.
- 📱 **Fully Responsive:** Looks great on desktop, tablet, and mobile devices.
- ⚡ **Fast & Optimized:** Built with React 19 and Vite for high performance.
- 🛠️ **Easy Customization:** Simple to rebrand with your organization's colors, logo, and content using Tailwind CSS.
- 🧩 **Form Handling:** Robust contact and donation forms powered by React Hook Form and Zod validation.
